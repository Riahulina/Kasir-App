# Kasir-App

Aplikasi Kasir berbasis mobile yang dikembangkan sebagai Project Based Learning (PBL) untuk memenuhi Ujian Akhir Semester (UAS).

## Fitur Utama

### Autentikasi Firebase

- Login menggunakan Firebase Authentication
- Registrasi akun menggunakan data dinamis
- Reset password (Forgot Password)
- Penyimpanan data pengguna secara real-time

### Manajemen Pengguna

- Registrasi operator kasir
- Pengelolaan profil pengguna
- Role pengguna sebagai kasir outlet

### Manajemen Produk

- Menambah produk
- Mengubah data produk
- Menghapus produk
- Menampilkan daftar produk

### Transaksi Kasir

- Menambahkan produk ke transaksi
- Perhitungan total pembayaran otomatis
- Riwayat transaksi
- Pengelolaan transaksi kasir

### Profil Pengguna

- Pengaturan profil
- Penyimpanan data profil pengguna

## Teknologi yang Digunakan

- React Native
- Expo
- TypeScript
- Firebase Authentication
- Firebase Firestore
- React Navigation

## Instalasi

```bash
git clone https://github.com/Riahulina/Kasir-App.git
cd Kasir-App
npm install
npx expo start
```
